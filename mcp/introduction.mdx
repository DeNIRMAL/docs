## What is MCP?

The Model Context Protocol (MCP) is an open standard that enables AI assistants to securely connect with data sources and tools. SuperMemory's MCP server allows you to use your saved knowledge directly within Claude Desktop and other MCP-compatible clients.

<Info>
MCP (Model Context Protocol) was created by Anthropic to standardize how AI assistants access external data and tools securely.
</Info>

## Installation

### Prerequisites

- Node.js 18 or higher
- A SuperMemory account with API access
- Claude Desktop (or another MCP client)

### Install via npm

```bash
npm install -g @supermemory/mcp-server
```

### Install from Source

```bash
git clone https://github.com/DeNIRMAL/memory2.git
cd memory2/packages/mcp-server
npm install
npm run build
npm link
```

## Configuration

### Claude Desktop Setup

Add SuperMemory to your Claude Desktop configuration:

<Tabs>
  <Tab title="macOS">
    Edit `~/Library/Application Support/Claude/claude_desktop_config.json`:
    ```json
    {
      "mcpServers": {
        "supermemory": {
          "command": "supermemory-mcp-server",
          "env": {
            "SUPERMEMORY_API_KEY": "your_api_key_here"
          }
        }
      }
    ```

### Advanced Configuration

You can customize the MCP server behavior:

```json
{
  "mcpServers": {
    "supermemory": {
      "command": "supermemory-mcp-server",
      "env": {
        "SUPERMEMORY_API_KEY": "your_api_key_here",
        "SUPERMEMORY_BASE_URL": "https://api.vevibe.com",
        "SUPERMEMORY_DEFAULT_SPACE": "your-default-space-uuid",
        "SUPERMEMORY_SEARCH_LIMIT": "20",
        "SUPERMEMORY_LOG_LEVEL": "info"
      }
    }
  }
}
```

### Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `SUPERMEMORY_API_KEY` | Your API key (required) | - |
| `SUPERMEMORY_BASE_URL` | API base URL | `https://api.vevibe.com` |
| `SUPERMEMORY_DEFAULT_SPACE` | Default space for new memories | User's first space |
| `SUPERMEMORY_SEARCH_LIMIT` | Default search result limit | `10` |
| `SUPERMEMORY_LOG_LEVEL` | Logging level | `warn` |

## Available MCP Tools

The SuperMemory MCP server provides these tools to Claude:

### search_memories

Search across your saved memories using semantic search.

**Parameters:**
- `query` (string): Search query
- `limit` (number, optional): Number of results (default: 10)
- `spaces` (array, optional): Space UUIDs to search within

**Example:**
```
search_memories({
  "query": "React hooks best practices",
  "limit": 5,
  "spaces": ["work-space-uuid"]
})
```

### add_memory

Add new content to SuperMemory.

**Parameters:**
- `type` (string): Content type ("page", "note", "document", "tweet")
- `content` (string): URL or text content
- `spaces` (array, optional): Space UUIDs to add to
- `title` (string, optional): Custom title

**Example:**
```
add_memory({
  "type": "note",
  "content": "Remember to review the quarterly reports",
  "spaces": ["work-space-uuid"],
  "title": "Q4 Review Reminder"
})
```

### list_spaces

Get all available spaces and their details.

**Parameters:**
None

**Example:**
```
list_spaces()
```

### get_space_contents

Get memories within a specific space.

**Parameters:**
- `space_id` (string): Space UUID
- `limit` (number, optional): Number of memories to return

**Example:**
```
get_space_contents({
  "space_id": "work-space-uuid",
  "limit": 20
})
```

### get_user_info

Get current user information and account details.

**Parameters:**
None

**Example:**
```
get_user_info()
```

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="MCP Server Not Connecting">
    **Symptoms:** Claude Desktop doesn't show SuperMemory tools
    
    **Solutions:**
    - Verify your API key is correct
    - Check that the MCP server is installed globally
    - Restart Claude Desktop after configuration changes
    - Check Claude Desktop logs for error messages
  </Accordion>

  <Accordion title="API Key Authentication Errors">
    **Symptoms:** "Unauthorized" or "Invalid API key" errors
    
    **Solutions:**
    - Generate a new API key from SuperMemory dashboard
    - Ensure the API key is properly set in the environment
    - Check that your account has API access enabled
  </Accordion>

  <Accordion title="Search Results Empty">
    **Symptoms:** Search returns no results despite having content
    
    **Solutions:**
    - Verify content has finished processing in SuperMemory
    - Try broader search terms
    - Check if you're searching in the correct spaces
    - Ensure your account has content in SuperMemory
  </Accordion>

  <Accordion title="Performance Issues">
    **Symptoms:** Slow responses or timeouts
    
    **Solutions:**
    - Reduce search limits in configuration
    - Check your internet connection
    - Verify SuperMemory API status
    - Try restarting the MCP server
  </Accordion>
</AccordionGroup>

## Advanced Usage

### Custom MCP Client

Build your own MCP client that uses SuperMemory:

```typescript
import { Client } from '@modelcontextprotocol/sdk/client/index.js';
import { StdioClientTransport } from '@modelcontextprotocol/sdk/client/stdio.js';

async function createSuperMemoryClient() {
  const transport = new StdioClientTransport({
    command: 'supermemory-mcp-server',
    env: {
      SUPERMEMORY_API_KEY: 'your_api_key_here'
    }
  });

  const client = new Client({
    name: 'my-supermemory-client',
    version: '1.0.0',
  }, {
    capabilities: {}
  });

  await client.connect(transport);
  return client;
}

// Use the client
const client = await createSuperMemoryClient();

// Search memories
const searchResult = await client.callTool({
  name: 'search_memories',
  arguments: {
    query: 'machine learning',
    limit: 5
  }
});

console.log(searchResult);
```

### Integration with Other Tools

Combine SuperMemory with other MCP servers:

```json
{
  "mcpServers": {
    "supermemory": {
      "command": "supermemory-mcp-server",
      "env": {
        "SUPERMEMORY_API_KEY": "your_api_key"
      }
    },
    "filesystem": {
      "command": "npx",
      "args": ["@modelcontextprotocol/server-filesystem", "/path/to/files"]
    },
    "brave-search": {
      "command": "npx",
      "args": ["@modelcontextprotocol/server-brave-search"],
      "env": {
        "BRAVE_API_KEY": "your_brave_api_key"
      }
    }
  }
}
```

This allows Claude to:
- Search the web with Brave
- Access local files
- Search your SuperMemory knowledge
- Cross-reference information from all sources
