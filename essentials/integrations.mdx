## Twitter Integration

### Features
- **Automatic Bookmark Sync**: Import all your Twitter bookmarks
- **Liked Tweet Capture**: Save tweets you've liked for later reference
- **Thread Reconstruction**: Automatically piece together Twitter threads
- **Real-time Sync**: New likes and bookmarks are automatically saved
- **Metadata Extraction**: Capture author, engagement metrics, and context

### Setup Process

<Steps>
  <Step title="Connect Your Account">
    Go to Settings → Integrations → Twitter and click "Connect Account"
  </Step>
  <Step title="Authorize Access">
    Log in to Twitter and authorize SuperMemory to access your bookmarks and likes
  </Step>
  <Step title="Configure Settings">
    Choose which content to sync and how often
  </Step>
  <Step title="Select Target Space">
    Pick which space to save Twitter content to
  </Step>
</Steps>

### Configuration Options

<Tabs>
  <Tab title="Content Types">
    - **Bookmarks**: Import all existing and new bookmarks
    - **Liked Tweets**: Save tweets you like to SuperMemory
    - **Your Tweets**: Backup your own tweets and threads
    - **Mentions**: Save tweets where you're mentioned
  </Tab>
  <Tab title="Sync Settings">
    - **Frequency**: Real-time, hourly, or daily sync
    - **Batch Size**: How many tweets to process at once
    - **Filtering**: Keywords or hashtags to include/exclude
    - **Date Range**: Import tweets from specific time periods
  </Tab>
  <Tab title="Organization">
    - **Default Space**: Where to save Twitter content
    - **Auto-tagging**: Automatically tag with #twitter, hashtags
    - **Thread Handling**: How to organize tweet threads
    - **Duplicate Prevention**: Avoid saving the same tweet multiple times
  </Tab>
</Tabs>

### Twitter API Usage

For advanced users with Twitter API access:

```bash
# Manually trigger Twitter sync
curl -X POST https://api.vevibe.com/v1/integrations/twitter/sync \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "bookmarks",
    "limit": 100
  }'
```


### Troubleshooting Notion Import

<Warning>
Large Notion workspaces may take several hours to import. You'll receive email notifications when the import is complete.
</Warning>

Common issues:
- **Permission Errors**: Ensure SuperMemory has access to the content you want to import
- **Rate Limiting**: Large imports may be throttled to respect Notion's API limits
- **Formatting Issues**: Some complex Notion formatting may not transfer perfectly
- **Missing Images**: Images in private Notion pages may not be accessible

## API Integrations

### Webhook Support
Set up webhooks to automatically sync content from custom sources:

```bash
# Configure webhook endpoint
curl -X POST https://api.vevibe.com/v1/integrations/webhooks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://your-webhook-endpoint.com",
    "events": ["content.created", "content.updated"],
    "secret": "your-webhook-secret"
  }'
```

### Custom Integrations
Build your own integrations using our API:

```javascript
// Example: RSS feed integration
const rssParser = require('rss-parser');
const parser = new rssParser();

async function syncRSSFeed(feedUrl) {
  const feed = await parser.parseURL(feedUrl);
  
  for (const item of feed.items) {
    await fetch('https://api.vevibe.com/v1/add', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        type: 'page',
        content: item.link,
        spaces: ['rss-feed-space-id'],
        prefetched: {
          title: item.title,
          description: item.contentSnippet,
          contentToSave: item.content,
        },
      }),
    });
  }
}
```

## Security and Privacy

### Authentication
All integrations use secure OAuth flows:
- **No Password Storage**: We never store your platform passwords
- **Limited Permissions**: Only request necessary permissions
- **Revocable Access**: Disconnect integrations anytime
- **Encrypted Transit**: All data transfer is encrypted

### Data Handling
- **Source Attribution**: Original source and author information is preserved
- **Permission Respect**: Private content remains private
- **Local Processing**: Content analysis happens in secure environments
- **Retention Control**: You control how long integrated content is stored

## Troubleshooting Integrations

### Common Issues

<AccordionGroup>
  <Accordion title="Integration disconnected">
    - Check if you've changed passwords on the source platform
    - Re-authorize the integration in SuperMemory settings
    - Verify the source platform hasn't revoked access
  </Accordion>

  <Accordion title="Content not syncing">
    - Check integration sync settings and frequency
    - Verify you have new content to sync on the source platform
    - Look for error messages in the integration logs
  </Accordion>

  <Accordion title="Missing content">
    - Some content may be private or restricted
    - Check date range settings for imports
    - Verify content types are enabled in sync settings
  </Accordion>

  <Accordion title="Duplicate content">
    - Our deduplication isn't perfect for heavily modified content
    - Use bulk operations to clean up duplicates
    - Adjust sync settings to prevent future duplicates
  </Accordion>
</AccordionGroup>
