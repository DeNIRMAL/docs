# Quickstart Guide

This guide will help you make your first API call to the Vevibe Memory API and create your first memory.

## Prerequisites

Before you begin, make sure you have:
- A Vevibe account (sign up at [vevibe.com](https://vevibe.com))
- An API key from your [dashboard](https://vevibe.com/dashboard)

## Step 1: Get Your API Key

1. Log in to your [Vevibe Dashboard](https://vevibe.com/dashboard)
2. Navigate to the API Keys section
3. Create a new API key or copy an existing one
4. Keep this key secure - you'll need it for all API requests

## Step 2: Create Your First Space

Spaces are containers that help organize your memories. Let's create one:

<CodeGroup>
```bash cURL
curl -X POST "https://api.vevibe.com/v1/spaces" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My First Space",
    "description": "A space for testing the Vevibe Memory API"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.vevibe.com/v1/spaces', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'My First Space',
    description: 'A space for testing the Vevibe Memory API'
  })
});

const space = await response.json();
console.log('Created space:', space);
```

```python Python
import requests

response = requests.post(
    'https://api.vevibe.com/v1/spaces',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'name': 'My First Space',
        'description': 'A space for testing the Vevibe Memory API'
    }
)

space = response.json()
print('Created space:', space)
```
</CodeGroup>

## Step 3: Add Your First Memory

Now let's add some content to your space:

<CodeGroup>
```bash cURL
curl -X POST "https://api.vevibe.com/v1/memories" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Vevibe Memory API is a powerful tool for building AI-driven applications with intelligent memory capabilities.",
    "type": "note",
    "spaceId": "SPACE_ID_FROM_STEP_2"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.vevibe.com/v1/memories', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    content: 'Vevibe Memory API is a powerful tool for building AI-driven applications with intelligent memory capabilities.',
    type: 'note',
    spaceId: 'SPACE_ID_FROM_STEP_2'
  })
});

const memory = await response.json();
console.log('Created memory:', memory);
```

```python Python
response = requests.post(
    'https://api.vevibe.com/v1/memories',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'content': 'Vevibe Memory API is a powerful tool for building AI-driven applications with intelligent memory capabilities.',
        'type': 'note',
        'spaceId': 'SPACE_ID_FROM_STEP_2'
    }
)

memory = response.json()
print('Created memory:', memory)
```
</CodeGroup>

## Step 4: Search Your Memories

Now let's search for the memory you just created:

<CodeGroup>
```bash cURL
curl -X GET "https://api.vevibe.com/v1/memories/search?q=Vevibe%20API&spaceId=SPACE_ID" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.vevibe.com/v1/memories/search?q=Vevibe%20API&spaceId=SPACE_ID', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const results = await response.json();
console.log('Search results:', results);
```

```python Python
response = requests.get(
    'https://api.vevibe.com/v1/memories/search',
    params={'q': 'Vevibe API', 'spaceId': 'SPACE_ID'},
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

results = response.json()
print('Search results:', results)
```
</CodeGroup>

## Step 5: Chat with Your Memories

The most powerful feature is the ability to have conversations with your stored data:

<CodeGroup>
```bash cURL
curl -X POST "https://api.vevibe.com/v1/chat" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "What did I learn about the Vevibe API?",
    "spaceId": "SPACE_ID"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.vevibe.com/v1/chat', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    message: 'What did I learn about the Vevibe API?',
    spaceId: 'SPACE_ID'
  })
});

const chatResponse = await response.json();
console.log('AI Response:', chatResponse.message);
```

```python Python
response = requests.post(
    'https://api.vevibe.com/v1/chat',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'message': 'What did I learn about the Vevibe API?',
        'spaceId': 'SPACE_ID'
    }
)

chat_response = response.json()
print('AI Response:', chat_response['message'])
```
</CodeGroup>
